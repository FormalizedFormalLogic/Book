### Compile

To compile this book, you need bookmd and some plugins.

- [mdbook](https://github.com/rust-lang/mdBook)
- [mdbook-katex](https://github.com/lzanini/mdbook-katex) for [KaTeX](https://katex.org/)

```shell
mdbook serve md

mdbook build md
```
